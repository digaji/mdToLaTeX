{\noindent \Large \textbf{Setting Up a Project}}\\\\
\\
Install the Angular CLI globally:\\
\\
\begin{lstlisting}
npm install -g @angular/cli
\end{lstlisting}
\\
Create workspace:\\
\\
\begin{lstlisting}
ng new [PROJECT NAME]
\end{lstlisting}
\\
Run the application:\\
\\
\begin{lstlisting}
cd [PROJECT NAME]
ng serve
\end{lstlisting}
